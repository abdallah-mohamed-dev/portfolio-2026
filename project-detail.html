The Challenge

Vanguard required a digital portal that could handle complex financial data visualizations while maintaining the ultra-premium feel of a private bank. The primary obstacle was integrating legacy API data into a modern, fast, and SEO-friendly WordPress environment.

Year 2026
Role Lead Frontend Architect
Stack Headless WP, Next.js, Framer Motion

01 // Dashboard architecture showing real-time asset tracking.

02

The Strategy

We built a custom GraphQL layer to bridge the legacy banking system with WordPress.

By decoupling the frontend from the backend, we achieved a perfect 100/100 Lighthouse score. The interface was designed using a "Glass-Brutalism" approach—combining sharp borders with soft, blurred translucent surfaces to represent transparency and security.
